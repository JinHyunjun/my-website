About

데이터의 구조를
이해하는 사람

실무에서 MariaDB · MySQL · OpenSearch 기반 운영 데이터를 직접 조회하고 조건을 정의하며, 고객 요구사항에 맞는 대시보드와 지표 구조를 기획해왔습니다.

데이터를 가공하고 분석한 뒤, 사용자가 이해하고 판단할 수 있는 형태로 설계하는 일을 좋아합니다. 시각화 엔지니어로서 쌓은 데이터 구조 이해와 화면 설계 경험을 바탕으로, 데이터 분석가(Data Analyst)와 데이터 엔지니어(Data Engineer)로 영역을 확장하고 있습니다.

특정 직무에 머무르지 않고 데이터의 수집 · 적재 · 분석 · 시각화 전 과정을 이해하는 사람으로서, 데이터가 의사결정의 중심이 되는 산업의 흐름 속에서 실질적인 가치를 만드는 역할로 성장하고자 합니다.
